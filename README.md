# jokeApi
Getting started with installing the jokeApi node module by [npm install](https://www.npmjs.com/package/npm-i) and make sure you have installed nodemon [npm install nodemon](https://www.npmjs.com/package//nodemon) command.

## Available scripts

In the project directory, you can run:

 ### `nodemon index.js`

 Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

## API documentation

You can read api documentation in [jokeApi](https://documenter.getpostman.com/view/6048123/2s9XxsTv8Y).
